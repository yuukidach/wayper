wayper

Cross-platform wallpaper manager with Wallhaven integration and AI-native control.

Download View on GitHub

Features

Wallhaven

Auto-downloads wallpapers based on your preferences. Browse and search Wallhaven online from the GUI.

Auto orientation

Portrait monitors get portrait wallpapers. No manual sorting.

Pool management

Validates, resizes, and rotates wallpapers automatically.

SFW / NSFW

One key to switch. Persistent across sessions.

History

Prev/next through wallpaper history per monitor.

Favorites & blacklist

Like/dislike with undo. Favorites stay in rotation.

Native GUI

Browse, search, filter, multi-select, drag-and-drop, right-click menus. GTK4 on Linux, AppKit on macOS.

AI-native

Built-in MCP server for Claude Code and other assistants.

JSON output

--json flag on every command for scripting.

Usage

wayper daemon start         # start background rotation + downloads
wayper daemon stop          # stop daemon
wayper next                 # next wallpaper
wayper prev                 # previous wallpaper
wayper fav [--open]         # favorite current wallpaper
wayper dislike              # blacklist + switch
wayper undislike            # undo last dislike
wayper mode [sfw|nsfw]      # toggle or set mode
wayper status               # show current state
wayper-gui                  # GUI app

GUI

wayper-gui — browse your pool, favorites, and disliked wallpapers with search, filter, and sort. Multi-select with Ctrl/Shift+click, drag-and-drop import, right-click context menus, preview zoom (Ctrl+scroll). Includes a Wallhaven online search tab, quick actions panel, and live daemon control with quota monitoring. GTK4 on Linux, AppKit on macOS.

1 2 3 switch category
Enter set as wallpaper
f favorite
x reject / restore
d delete
o open on Wallhaven
m toggle SFW/NSFW
/ search
Ctrl+A select all
Esc clear search / deselect

MCP

Built-in MCP server lets AI assistants control your wallpapers. Ask Claude to "favorite this wallpaper" or "switch to something with mountains" — it just works.

{
  "mcpServers": {
    "wayper": {
      "command": "/path/to/.venv/bin/wayper-mcp"
    }
  }
}

Tools: status next_wallpaper prev_wallpaper fav unfav dislike undislike set_mode delete_wallpaper

Requirements

  • Python 3.12+
  • Wallhaven API key
  • Linux: awww, Hyprland, GTK4 + PyGObject
  • macOS: PyObjC (pip install 'wayper[macos]')
