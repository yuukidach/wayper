wayper

Cross-platform wallpaper manager with Wallhaven integration and AI-native control.

Download GitHub AUR

Features

Wallhaven
Auto-downloads wallpapers based on your search preferences.
Auto orientation
Portrait monitors get portrait wallpapers. No manual sorting.
Pool management
Validates, resizes, and rotates wallpapers automatically.
SFW/NSFW toggle
One key to switch. Persistent across sessions.
History
Prev/next through wallpaper history per monitor.
Favorites & blacklist
Like/dislike with undo. Favorites stay in rotation.
Native browser
GTK4 on Linux, AppKit on macOS. Keyboard-driven.
AI-native
Built-in MCP server for Claude Code and other AI assistants.
JSON output
--json flag on every command for scripting.

Install

Arch Linux (AUR)

paru -S wayper     # or: yay -S wayper

From source

git clone https://github.com/yuukidach/wayper.git
cd wayper
uv venv && uv pip install -e .

macOS

pip install 'wayper[macos]'

Usage

wayper daemon               # start background rotation + downloads
wayper next                 # next wallpaper
wayper prev                 # previous wallpaper
wayper fav [--open]         # favorite current wallpaper
wayper dislike              # blacklist + switch
wayper undislike            # undo last dislike
wayper mode [sfw|nsfw]      # toggle or set mode
wayper status               # show current state
wayper browse               # native wallpaper browser

Browse

1 2 3 switch category
Enter set as wallpaper
f favorite
x remove / reject / restore
o open on Wallhaven
d delete
m toggle SFW/NSFW
q / Esc close

MCP

Built-in MCP server lets AI assistants control your wallpapers natively.

{
  "mcpServers": {
    "wayper": {
      "command": "/path/to/.venv/bin/wayper-mcp"
    }
  }
}

Tools: status next_wallpaper prev_wallpaper fav unfav dislike undislike set_mode delete_wallpaper

Requirements

  • Python 3.12+
  • Wallhaven API key
  • Linux: swww, Hyprland, GTK4 + PyGObject
  • macOS: PyObjC (pip install 'wayper[macos]')
